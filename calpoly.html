English 412 New Media Arts II

In English 412, I created a "Netflix Movie Generator." After pressing the button, the code will randomly select a mood and a movie on Netflix that will fit whatever mood you are feeling.

I also created an Interactive Text Adventure Game. using primarily JavaScript. The user is in control of their own fate and must find the right path towards success. If a wrong selection is made the user is forced to restart the game until they are able to finish the game.
